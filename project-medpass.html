Project Case Study

MedPass - Medical Clearance Status Management System

A web-based system designed to streamline and digitize the medical clearance process for students, allowing efficient tracking, record management, and role-based access for administrators and patients.

GitHub Video Demo Back to Projects

Project overview

What it is, who it's for, and the outcome.

  • Context: Academic project focused on healthcare workflow automation
  • Role: Full-stack developer (Admin Dashboard, Patient Management, Search functionality)
  • Timeline: Team project (3 members)
  • Impact: Streamlined medical clearance tracking, improved record organization, enabled real-time status updates

Technologies used

Technical stack and development tools.

PHP MySQL HTML CSS JavaScript XAMPP

Problem

The challenge we addressed in medical clearance management.

Manual tracking of medical clearance processes leads to delays, disorganized records, and inefficient communication between patients and administrators. Students struggled to track their clearance status, while staff faced challenges in managing and updating records efficiently.

Solution

Our approach and the system we built to solve the medical clearance workflow.

  • Designed a centralized web-based system for real-time tracking and management
  • Implemented role-based access control for patients and administrators
  • Created intuitive dashboards for status monitoring and record updates
  • Built search functionality for efficient record retrieval

Key Features

Core functionality and system capabilities.

  • User authentication with role-based access (Patient & Admin)
  • Patient dashboard with real-time clearance status tracking
  • Admin dashboard for comprehensive record management
  • Add, update, and view patient information with database integration
  • Search functionality for efficient record retrieval
  • Organized and centralized record system

Challenges

Obstacles we encountered and how we overcame them.

  • Frontend-backend integration: Successfully connected UI with PHP and MySQL database
  • Role-based access control: Implemented secure authentication and permission systems
  • Database optimization: Designed efficient queries and normalized table structures
  • UI/UX design: Created intuitive interfaces for both patient and admin workflows

  • Team collaboration and task coordination in development projects
  • Problem-solving in healthcare workflow automation
